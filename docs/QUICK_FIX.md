# Быстрое исправление проблем

## Критические исправления (обязательно)

### 1. Добавить OPENROUTER_API_KEY

Откройте `.env` и добавьте:
```env
OPENROUTER_API_KEY=your_openrouter_api_key_here
```

**Как получить ключ:**
1. Зарегистрируйтесь на https://openrouter.ai
2. Перейдите в Settings → Keys
3. Создайте новый ключ
4. Скопируйте и вставьте в `.env`

**Без этого:** AI анализ работает только через keyword fallback (низкая точность)

---

### 2. Добавить ADMIN_TELEGRAM_IDS

Откройте `.env` и добавьте:
```env
ADMIN_TELEGRAM_IDS=ваш_telegram_id
```

**Как узнать свой Telegram ID:**
1. Напишите боту @userinfobot в Telegram
2. Он покажет ваш ID (число, например: 123456789)
3. Скопируйте ID и вставьте в `.env`

**Для нескольких админов:**
```env
ADMIN_TELEGRAM_IDS=123456789,987654321,555666777
```

**Без этого:** Админ-панель недоступна (команда `/admin` не работает)

---

### 3. Проверить Firebase подключение

Проверьте доступность прокси:
```bash
curl https://anthropic-proxy.uiredepositionherzo.workers.dev/firebase
```

Если не работает:
- Проверьте интернет-соединение
- Проверьте настройки прокси в `core/http_client.py`
- Убедитесь, что Cloudflare Worker активен

---

## Улучшения производительности (рекомендуется)

### Установить cryptg для ускорения

```bash
pip install cryptg
```

**Эффект:** Ускорит работу с Telegram API в 2-3 раза

---

## После исправлений

1. Перезапустите бота:
   ```bash
   py start_telegram_bot.py
   ```

2. Проверьте работу:
   - Отправьте `/admin` в боте (должна открыться админ-панель)
   - Создайте тестовую жалобу (должен работать AI анализ)

3. Запустите проверку системы:
   ```bash
   py test_system.py
   ```

---

## Проверка статуса

После исправлений все должно быть ✅:
- ✅ OPENROUTER_API_KEY: задан
- ✅ ADMIN_TELEGRAM_IDS: задан
- ✅ Firebase: подключается
- ✅ Админ-панель: работает
