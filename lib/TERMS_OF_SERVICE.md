# Условия использования

**Дата вступления в силу:** 9 февраля 2026 г.
**Версия:** 1.0.0

---

## 1. Общие положения

Используя приложение "СообщиО", вы соглашаетесь с настоящими Условиями использования.

---

## 2. Описание приложения

Приложение "СообщиО" - сервис для подачи и отслеживания городских проблем (жалоб, замечаний, предложений) в городе Нижневартовск.

---

## 3. Правила использования

### 3.1. Регистрация и авторизация

1. **Регистрация через Telegram:**
   - Для использования всех функций приложения необходимо авторизоваться через Telegram
   - Авторизация безопасна и не требует ввода пароля
   - При авторизации приложение запрашивает доступ к:
     * Имени пользователя
     * Фотографии профиля
     * Контактов

2. **Возрастные ограничения:**
   - Приложение предназначено для пользователей старше 14 лет
   - Пользователи младше 14 лет могут использовать приложение только с согласия родителей

3. **Аккаунт:**
   - Каждый пользователь имеет право на один аккаунт
   - Создание множества аккаунтов запрещено
   - Попытки создания фейковых аккаунтов приведут к блокировке

---

### 3.2. Создание жалоб

1. **Правила подачи жалоб:**
   - Подавать можно только правдивую информацию
   - Не публиковывать оскорбительные или непристойный контент
   - Не публиковывать персональные данные третьих лиц
   - Не публиковывать фотографии других людей без их согласия
   - Указывать максимально точную информацию о проблеме

2. **Запрещенный контент:**
   - Призывы оскорбительного характера
   - Разжигание национальной, расовой или религизной ненависти
   - Призывы сексуального характера
   - Контент, нарушающий законодательство РФ
   - Мошенничество или фальшивые жалобы

3. **Фотографии:**
   - Фотографии должны быть четкими и понятными
   - Нельзя публиковывать фотографии с фильтрами (исключение: коррекция освещения, цвета)
   - Запрещено публиковывать фотографии несовершеннолетних без их согласия

4. **Категории жалоб:**
   - ЖКХ (водоснабжение, отопление, электричество, мусор)
   - Дороги (ямы, выбоины, дорожное полотно)
   - Освещение (неработающие уличные фонари, темные улицы)
   - Транспорт (маршруты, расписания)
   - Экология (загрязнение воздуха, выгул собак)
   - Образование (отсутствие школьной программы, проблемы с воспитанием)
   - Безопасность (опасные участки, нехватка освещения)
   - МЧС (нелегальная торговля, проблемы в общественных местах)
   - Связь (невыходящие связи, проблемы с интернетом)
   - Другие

5. **Обработка жалоб:**
   - Жалоба обрабатывается в течение 7 дней
   - Пользователь получает уведомление об изменении статуса
   - Пользователь может добавить дополнительные информацию
   - Пользователь может закрыть жалобу при ее решении
   - Администрация имеет право отклонить жалобу

6. **Ответственность:**
   - Пользователь несет ответственность за достоверность информации
   - Неправомерные или ложные жалобы могут быть отклонены
   - Предумеренные жалобы ведут к блокировке аккаунта

---

### 3.3. Комментрии и лайки

1. **Комментрии:**
   - Коментарии должны быть вежливы и конструктивными
   - Запрещены оскорбления, угрозы, ненормативная лексика
   - Комментарии, нарушающие законодательство, будут удалены
   - Коментарии, призывающие к противоправной деятельности, запрещены

2. **Лайки:**
   - Лайки выражают согласие с жалобой
   - Можно ставить лайк любой жалобе
   - Лайки можно убрать в любой момент
   - Лайки не влияют на рассмотрение жалобы

3. **Отзывы:**
   - Отзывы должны быть объективными и правдивыми
   - Запрещены угрожающие или клеветальные отзывы
   - Мошеннические отзывы удаляются

---

### 3.4. Защита данных и конфиденциальность

1. **Безопасность:**
   - Все данные защищены современными методами шифрования
   - Передача данных осуществляется по защищенным каналам (HTTPS)
   - Пароли не хранятся (используется токен авторизации)

2. **Права на удаление:**
   - Пользователь может удалить свои данные и жалобы
   - Удаление аккаунта невозможно в приложении
   - Для удаления аккаунта необходимо обратиться в поддержку

3. **Доступ к данным:**
   - Пользователь имеет право получить все свои данные
   - Данные могут быть экспортированы в формате JSON
   - Приложение может запросить доступ к геолокации
   - Приложение может запросить доступ к камере

4. **Данные, не собираемые:**
   - Приложение не собирает историю просмотра
   - Приложение не собирает историю поиска
   - Приложение не собирает историю нажатий

---

### 3.5. Уведомления

1. **Push-уведомления:**
   - Приложение может отправлять push-уведомления
   - Уведомления отправляются об изменении статуса жалобы
   - Уведомления отправляются о новых жалобах в вашем районе
   - Уведомления можно отключить в настройках

2. **Типы уведомлений:**
   - Статус жалобы изменен
   - Ваша жалоба получила комментарий
   - Ваша жалоба закрыта
   - Новые жалобы в вашем районе

3. **Настройки уведомлений:**
   - Push-уведомления включены по умолчанию
   - Пользователь может отключить уведомления
   - Пользователь может выбрать типы уведомлений

---

### 3.6. Противодействие с администрацией

1. **Обращения к поддержке:**
   - Обращения принимаются через Telegram бот или email
   - Время ответа: до 7 дней
   - Срочные жалобы рассматриваются в течение 24 часов

2. **Блокировки и санкции:**
   - За нарушение правил использования аккаунт блокируется
   - За нарушение правил поведения аккаунт блокируется
   - Срок блокировки: в зависимости от тяжести нарушения
   - За непрерывные нарушения аккаунт удаляется

---

### 3.7. Отказ в обслуживании

1. **Основания для отказа:**
   - Нарушение правил использования
   - Многократное нарушение правил
   - Систематическая подача ложных жалоб
   - Угроза администрации или других пользователей
   - Использование приложения для противоправной деятельности

2. **Процедура отказа:**
   - Пользователь получает уведомление
   - Указывается причина отказа
   - Дается возможность апелляции
   - Приложение блокируется при повторных нарушениях

---

## 4. Изменения в Условиях

1. **Внесение изменений:**
   - Приложение может вносить изменения в настоящие Условия
   - Об изменениях пользователи будут уведомляться
   - Продолжение использования означает согласие с изменениями

2. **Прекращение использования:**
   - Условия действуют до момента их отмены
   - При отмене пользователь может удалить свои данные в течение 30 дней
   - После удаления данные не восстанавливаются
